## 구현 기능
### 입력
- [ ] 코치 이름 입력
  - [ ] 코치는 2~5명 사이로 입력 된다.
  - [ ] 코치 이름은 2~4글자 
  - [ ] ',' 로 구분

- [ ] 못먹는 음식 입력
  - [ ] 먹지 못하는 메뉴는 0 ~ 2개
  - [ ] ',' 로 구분

- [ ] 잘못된 값을 입력하면, 예외 문구를 출력하고 다시 입력 받는다.

### 출력
- [ ] 서비스 시작 문구
- [ ] 서비스 종료 문구
- 
- [ ] 예외 상황 시 '[ERROR]' 로 시작하는 문구 출력

### 기능
- [ ] 카테고리를 결정하는 기능
  - [ ] 3회 이상 중복되지 않는 카테고리 
- [ ] 메뉴를 추천하는 기능
  - [ ] 한 주에 같은 음식을 중복해서 먹지 않음
